# SmartWaste Pro ♻️

## 📌 Project Overview
SmartWaste Pro is an IoT-based intelligent waste monitoring and marketplace system.

It uses a smart bin with 3 compartments:
- Plastic
- Food (Organic)
- Metal

Each compartment is monitored using sensors and connected to a cloud system.

The system connects:
- Waste suppliers (bin owners)
- Waste buyers (recyclers)

## 🎯 Objective
- Monitor waste levels in real-time
- Reduce waste overflow
- Support recycling economy
- Connect buyers and suppliers

## ⚙️ Key Features
- Real-time bin monitoring (ESP32 + Sensors)
- Firebase cloud integration
- Smart alerts (Buzzer + App)
- Buyer-Seller marketplace
- Automated workflows (n8n)

## 🧠 System Workflow
1. Sensors detect waste level
2. ESP32 processes data
3. Data sent to Firebase
4. n8n triggers automation
5. Buyers get notifications
6. Owner selects best offer

## 🧩 Technologies Used
- ESP32
- Ultrasonic Sensors
- Firebase
- n8n / Node-RED
- Mobile/Web App
